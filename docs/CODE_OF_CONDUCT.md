# Code of Conduct

## Our Pledge

We as contributors and maintainers pledge to make participation in our project and community a harassment-free experience for everyone, regardless of age, body size, disability, ethnicity, gender identity and expression, level of experience, nationality, personal appearance, race, religion, or sexual identity and orientation.

## Our Standards

### Positive behavior includes:

- Using welcoming and inclusive language
- Being respectful of differing viewpoints and experiences
- Gracefully accepting constructive criticism
- Focusing on what is best for the community and the project
- Showing empathy toward other community members

### Unacceptable behavior includes:

- Trolling, insulting/derogatory comments, and personal or political attacks
- Public or private harassment
- Publishing others' private information without explicit permission
- Spamming, self-promotion, or off-topic content in Issues/PRs
- Any conduct which could reasonably be considered inappropriate in a professional setting

## Auto-Ignore Policy

Issues and Pull Requests will be **automatically closed without response** if they:

1. **Contain suspicious code or links** -- PRs with potential malware will be reviewed in a sandboxed virtual machine. If malicious code is confirmed, the contributor will be permanently banned.
2. **Discuss unrelated topics** -- This project is an auto-click utility. Off-topic discussions (politics, other software, etc.) belong elsewhere.
3. **Include phishing or data-theft links** -- Any link suspected of stealing user data will result in immediate closure and a ban.
4. **Are excessively verbose or vague** -- Rambling descriptions without clear points, or descriptions so generic they could apply to any software, will be closed. Be specific and concise.
5. **Insult or disrespect any contributor** -- Zero tolerance for personal attacks, whether directed at the maintainer, other contributors, or AI tools used in development.

## Scope

This Code of Conduct applies within all project spaces, including GitHub Issues, Pull Requests, Discussions, and any other communication channel associated with this project.

## Enforcement

Instances of abusive, harassing, or otherwise unacceptable behavior may be reported by opening a private message to the maintainer ([@poli0981](https://github.com/poli0981)). All complaints will be reviewed and investigated.

The maintainer reserves the right to remove, edit, or reject any contributions that violate this Code of Conduct, and to temporarily or permanently ban any contributor for behavior deemed inappropriate, threatening, or harmful.
